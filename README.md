# 🛒 E-Commerce Sales Analytics & Business Insights

## 📊 Project Overview
This project performs Exploratory Data Analysis (EDA) on an E-commerce dataset containing 10,000 orders.

The goal is to uncover business insights related to:
- Customer purchasing behavior
- Sales trends over time
- Platform and city-wise performance
- Revenue distribution

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🧹 Data Cleaning
- Checked missing values
- Removed duplicates
- Converted date columns into proper datetime format

---

## 🧠 Feature Engineering
- Extracted Order Month & Year
- Created Price Categories (Low, Medium, High)

---

## 📈 Key Insights
- Identified top-performing cities
- Found peak sales months
- Analyzed customer purchase patterns
- Compared platform performance

---

## 📊 Visualizations
- Sales trends over time
- Category-wise revenue
- City-wise distribution
- Platform comparison charts
